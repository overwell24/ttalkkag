# 딸깍
생성형 AI((LLM)을 활용한 유튜브 영상 요약 서비스

## 시스템 아키텍처
### map & reduce 요약 방식
긴 영상을 여러 부분으로 나눈 후(map 단계), 각 부분을 개별적으로 요약하고 이를 하나의 요약본으로 통합(reduce 단계)하는 방식입니다.<br>
이 과정을 통해 긴 영상도 자막을 기반으로 효율적으로 요약할 수 있어, 사용자에게 핵심 정보를 빠르고 편리하게 제공합니다.
